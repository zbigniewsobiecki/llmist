The **llmist CLI** brings the power of AI agents directly to your terminal. Run completions, interactive agents with tools, generate images and speech, and create powerful automation workflows - all from the command line.

## Why Use the CLI?

<CardGrid>
  <Card title="Quick Completions" icon="rocket">
    Get AI responses without writing any code - pipe input, get output
  </Card>
  <Card title="Interactive Agents" icon="puzzle">
    Run agents with gadgets and MCP tools in an interactive TUI
  </Card>
  <Card title="Automation" icon="seti:pipeline">

  </Card>
  <Card title="Custom Commands" icon="setting">
    Define reusable commands in TOML configuration
  </Card>
</CardGrid>

## Key Features

- **Built-in commands** - complete, agent, image, speech, vision, models, gadget, mcp, init, config
- **TUI mode** - Beautiful terminal interface with expandable blocks and keyboard navigation
- **Gadget ecosystem** - Load gadgets from local files, npm packages, or git repositories
- **MCP integration** - Attach external MCP servers, import Claude Code config, or publish llmist gadgets via `llmist mcp serve`
- **TOML configuration** - Define custom commands, templates, and workflows
- **Gadget approval** - Security controls for tool execution

## Quick Example

```bash
    # Simple completion
    npx @llmist/cli complete "What is the capital of France?"

    # Interactive agent (TUI activates automatically in terminals)
    npx @llmist/cli agent "Help me organize my project"

    # Agent with an MCP filesystem server
    npx @llmist/cli agent \
      --mcp-server fs="npx -- -y @modelcontextprotocol/server-filesystem /tmp" \
      "List files in /tmp"

    # Generate an image
    npx @llmist/cli image "A sunset over mountains" -o sunset.png

    # Custom command from config
    npx @llmist/cli code-review ./src/main.ts
    ```

<Aside type="tip">
  The CLI automatically discovers API keys from environment variables (`OPENAI_API_KEY`, `ANTHROPIC_API_KEY`, `GEMINI_API_KEY`, `HF_TOKEN`).
</Aside>
